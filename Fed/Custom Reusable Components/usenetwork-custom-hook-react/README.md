# Display Network is Online or Offline


### Purpose

- Display user online and offilne based on their network connection.

### prerequisite:

- Node version: v18.18.2
- NPM version: 9.8.1

### Usage & Installing Steps

- Step 1:
  ```
  npm install
  ```
- Step 2:
  ```
  npm run start
  ```
