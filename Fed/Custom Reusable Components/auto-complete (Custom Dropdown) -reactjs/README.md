# Custom Auto Complete Dropdown

### prerequisite:

- Node version: v16.16.0
- NPM version: 9.6.7

#### Setup Project

```bash
$ npm i
$ npm run start
```

> open a `localhost:3000` in your browser

#### Build Command

```bash
$ npm run build
```
